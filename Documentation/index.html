Jestem is an elegant & minimal personal resume vCard HTML / CSS theme based on Bootstrap 3 framework!. Thanks to responsive design it is going to look and work awesome on any device!

HTML Structure

Whole template is based on Bootstrap 3 framework. Everything about this framework you will find here.

Below you can see the structure of the main page HTML file:

PHP Code

This template uses one PHP script:

1. contact-form.php - You need to config it to recieve messages from you website.


    $emailTo = 'example@mail.com'; //Put your own email address here
    $subject = 'Message from your website.';

Sendmail is not installed by default on a lot of servers so the form works so long as sendmail is installed.

3. twitter/index.php - Twitter Feed configuration file

    // Your Twitter App Consumer Key
    private $consumer_key = 'YOUR KEY';

    // Your Twitter App Consumer Secret
    private $consumer_secret = 'YOUR KEY';

    // Your Twitter App Access Token
    private $user_token = 'YOUR KEY';

    // Your Twitter App Access Token Secret
    private $user_secret = 'YOUR KEY';

To get those codes you need to generate it in your Twitter Account API

Navigation

Navigation is visible on each sub-view and may display user avatar, user name, page caption. There are also such elements like navigation buttons and navigation trigger which opens main menu window.

This html part of code includes user name and page caption:

<h2 class="margin-bottom-0">Mary Jane <strong>Doe</strong></h2>
<h6 class="nav-caption">I’m 28 years old proffesional <span class="text-primary">mental coach</span> from Poland.</h6>

There is possibility to view page navigation on each view (even start view) by adding visible-on-start class to #main-nav element.

User name is static and it is placed in <h2></h2> selector.

<h2 class="margin-bottom-0">Mary Jane <strong>Doe</strong></h2>

Default page caption is defined in <h6></h6> selector.

<h6 class="nav-caption">I’m 28 years old proffesional <span class="text-primary">mental coach</span> from Poland.</h6>

This is dynamic element and it changes after switching visible section on the viewport. It is connected with data-caption attribute of the each section:

<section id="resume" data-caption="Check my <span>skills</span> &amp; <span>abilities</span>!" data-title="My Resume">

Content buttons are dynamic elements and they are defined in this html code:

<div class="content-buttons">
   <a href="#" class="btn btn-primary" data-target="next-section" data-placement="0">Check my resume<i class="icon-after fa fa-chevron-right"></i></a>
   <a href="#" class="btn btn-primary" data-placement="1">Download my CV</a>
   <a href="blog.html" class="btn btn-primary" data-placement="3">Check my Blog</a>
</div>

To place buttons in navigation content it is required to set section number where the button should display (data-placement attribute - 0 is the start view).

<a href="#" class="btn btn-primary" data-placement="1">Download my CV</a>

 

Google Map

To configure your location you have to edit two lines in js/GoogleMap.js file

var yourLatitude = 40.758895;
var yourLongitude = -73.985131;

You need to use special toolto find your latitude and longitude (f.e http://itouchmap.com/latlong.html)

To change GoogleMap theme please change data-style attribute in GoogleMap selector:

<div id="google-map" data-style="dark"></div>

There is available 4 options - light, grey, standard and dark.

 

Pie Chart

Pie Chart is available in two size versions:

- Large

<div class="chart chart-lg" data-bar-color="#7bc82f" data-track-color="#e5e5e5" data-percent="75"><span class="percent"></span></div>

- Small

<div class="chart chart-sm" data-bar-color="#7bc82f" data-track-color="#e5e5e5" data-percent="25"><span class="percent"></span></div>

To set color of this chart please change data-bar-color and data-track-color attributes.

 

Page Loader

To turn off page loader remove this HTML code:

<!-- Loader -->
<div id="page-loader" class="loader-white"></div>

this JS part from custom.js:

Pace.on("done", function() {
            $('.pace-progress, .pace').delay(1000).fadeOut(100, function() {
                $('#page-loader').delay(300).fadeOut(700);
            });
        });

and this JS Script from each HTML file:

<script type="text/javascript" src="js/page-loader.js"></script>

 

Element Alpha

To customize transaprency of your photo you need to use similar code like in the background image in #profile section

<img alt="" src="logo.jpg" class="editable-alpha" data-alpha="40">

.editable-alpha - required class

data-alpha="image opacity in %" - required attribiute

 

Sticky Nav

If you want to turn off sticky navigtaion please remove sticky-nav class from <body> selector.
